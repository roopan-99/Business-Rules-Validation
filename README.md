# 📊 Business Rules Validation & Reporting Analysis

## 📌 Overview

This project demonstrates how Microsoft Excel and SQL can be used together to validate business rules, improve data quality, and generate meaningful business reports.

The dataset contains customer and transaction information that was validated using predefined business rules. SQL queries were used to analyze the data and create summary reports.
